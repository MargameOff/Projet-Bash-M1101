Pour tester le programme veuillez suivre les instructions suivantes :


-Ouvrir le terminal (Linux: Terminal, Windows: Cygwin(http://cygwin.com/setup-x86_64.exe))

-Rendez-vous sur le répertoire du logiciel :

--> cd <Répertoire>
(exemple): --> cd /home/projet-fr

-Lancez le programme avec la commande ci-suivante :

./Launcher.sh


-selectionner la langue:-English
			-Français
			-中文
			-Beutsche
			-España
			-Italiano
			-Latine

-Taper le chiffre correspondant à choix des actions
